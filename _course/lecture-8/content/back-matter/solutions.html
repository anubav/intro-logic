Exercise 1.4

  1. Suppose \(X\subset Y\), and let \(a\in X\cup Z\). Then, either \(a\in X\) or \(a\in Z\). If, on the one hand, \(a\in X\), then, since \(X\subset Y\), it follows that \(a\in Y\) and so \(a\in Y\cup Z\). If, on the other hand, \(a\in Z\), then \(a\in Y\cup Z\). So, in either case, \(a\in Y\cup Z\). Hence, if \(a\in X\cup Z\), then \(a\in Y\cup Z\), that is \(X\cup Z\subset Y\cup Z\).

  3. First, suppose \(X\subset Y^c\) and let \(a\in Y\). Then \(a\not\in Y^c\), and so, since \(X\subset Y^c\), it follows that \(a\not\in X\). So \(a\in X^c\). Hence, if \(a\in Y\), then \(a\in X^c\), that is, \(Y\subset X^c\).

    Next, suppose \(Y\subset X^c\) and let \(a\in X\). Then \(a\not\in X^c\), and so, since \(Y\subset X^c\), it follows that \(a\not\in Y\). So \(a\in Y^c\). Hence, if \(a\in X\), then \(a\in Y^c\), that is, \(X\subset Y^c\).

  4. First, suppose \(a\in X\cup(Y-X)\). Then, either \(a\in X\) or \(a\in Y-X\). If, on the one hand, \(a\in X\), then \(a\in X\cup Y\). If, on the other hand \(a\in Y-X\), then \(a\in Y\), and so \(a\in X\cup Y\). So, in either case, \(a\in X\cup Y\). Hence, if \(a\in X\cup(Y-X)\), then \(a\in X\cup Y\), that is \(X\cup(Y-X)\subset X\cup Y\).

    Next, suppose \(a\in X\cup Y\). Then, either \(a\in X\) or \(a\in Y\). If, on the one hand, \(a\in X\), then \(a\in X\cup(Y-X)\). Suppose, on the other hand, that \(a\in Y\). If \(a\in X\), then \(a\in X\cup(Y-X)\). If \(a\not\in X\), then \(a\in Y-X\), so again \(a\in X\cup(Y-X)\). So, in either case, \(a\in X\cup(Y-X)\). Hence, if \(a\in X\cup Y\), then \(a\in X\cup(Y-X)\), that is \(X\cup Y\subset X\cup(Y-X)\).

    Since \(X\cup(Y-X)\subset X\cup Y\) and \(X\cup Y\subset X\cup(Y-X)\), it follows that \(X\cup(Y-X)=X\cup Y\).

Exercise 1.9

Suppose that \(\Gamma,\varphi\models_V\psi\) and \(\Delta\models_V\varphi\), and let \(v\in V\) be a truth vuation such that \(v(\chi)=\mathbf{T}\) for all \(\chi\in \Gamma\cup\Delta\). Since \(\Delta\subset \Gamma\cup\Delta\) it follows that \(v(\chi)=\mathbf{T}\) for all \(\chi\in\Delta\). So, since \(\Delta\models_V\varphi\), we have \(v(\varphi)=\mathbf{T}\). Moreover, since \(\Gamma\subset \Gamma\cup\Delta\) it follows that \(v(\chi)=\mathbf{T}\), for all \(\chi\in\Gamma\). Hence, \(v(\chi)=\mathbf{T}\), for all \(\chi\in\Gamma\cup\{\varphi\}\). So, since \(\Gamma\cup\{\varphi\}\models_V\psi\), \(v(\psi)=\mathbf{T}\). Hence, any valuation in \(V\) under which all of the sentences in \(\Gamma\cup\Delta\) are true is one in which \(\psi\) is true. In other words, \(\Gamma\cup\Delta\models_V\psi\).

Exercise 1.11

  1. \(B\textrm{\textbf{i}}A, B\textrm{\textbf{a}}C\models A\textrm{\textbf{i}}C\)

    Let \(\sigma\) be an interpretation on which both of the premises are true. Since \(B\textrm{\textbf{i}}A\) is true on this intepretation, \(\sigma(B)\cap\sigma(A)\neq\emptyset\). Choose some \(a\in\sigma(B)\cap\sigma(A)\). Then, \(a\in\sigma(B)\). But since \(B\textrm{\textbf{a}}C\) is true, \(\sigma(B)\subset\sigma(C)\), and so \(a\in\sigma(C)\). But since \(a\in\sigma(B)\cap\sigma(A)\), it is also the case that \(a\in\sigma(A)\). Hence \(a\in\sigma(A)\cap\sigma(C)\). But then \(\sigma(A)\cap\sigma(C)\neq\emptyset\). So \(A\textrm{\textbf{i}}C\) is true on the interpretation \(\sigma\) as well. Hence, the argument is valid.

  2. \(B\textrm{\textbf{e}}A,B\textrm{\textbf{i}}C\not\models A\textrm{\textbf{o}}C\)

    Let \(a,b,c\in\Omega\) and let \(\sigma\) be an interpretation such that: \[ \sigma(A) = \{a \} \]

    \[ \sigma(B) = \{b, c\} \]

    \[ \sigma(C) = \{a, b\} \]

    On this interpretation:

    \[ \sigma(B)\cap\sigma(A) = \{b, c\}\cap\{a\} = \emptyset \hspace{3mm}\Rightarrow\hspace{3mm} v^\sigma(B\textrm{\textbf{e}}A) = \mathbf{T} \]

    \[ \sigma(B)\cap\sigma(C) = \{b, c\} \cap\{a,c\} = \{a\} \hspace{3mm}\Rightarrow\hspace{3mm} v^\sigma(B\textrm{\textbf{i}}C) = \mathbf{T} \]

    \[ \sigma(A)-\sigma(C) = \{a \}-\{a, b\} = \emptyset \hspace{3mm}\Rightarrow\hspace{3mm} v^\sigma(A\textrm{\textbf{o}}C) = \mathbf{F} \]

    Hence, \(\sigma\) is a counterexample to the argument, and so the argument is invalid.

  4. \(A\textrm{\textbf{a}}B, C\textrm{\textbf{a}}B, B\textrm{\textbf{a}}D\not \models A\textrm{\textbf{i}}C\)

    Let \(\sigma\) be an interpretation such that: \[ \sigma(A) = \emptyset \]

    \[ \sigma(B) = \emptyset \]

    \[ \sigma(C) = \emptyset \]

    On this interpretation:

    \[ \sigma(A) = \emptyset \subset \sigma(B) \hspace{3mm}\Rightarrow\hspace{3mm} v^\sigma(A\textrm{\textbf{a}}B) = \mathbf{T} \]

    \[ \sigma(C) = \emptyset \subset \sigma(B) \hspace{3mm}\Rightarrow\hspace{3mm} v^\sigma(C\textrm{\textbf{a}}B) = \mathbf{T} \]

    \[ \sigma(B) = \emptyset \subset \sigma(D) \hspace{3mm}\Rightarrow\hspace{3mm} v^\sigma(B\textrm{\textbf{a}}D) = \mathbf{T} \]

    \[ \sigma(A)\cap\sigma(C) = \emptyset \cap\emptyset = \emptyset \hspace{3mm}\Rightarrow\hspace{3mm} v^\sigma(A\textrm{\textbf{i}}C) = \mathbf{F} \]

    Hence, \(\sigma\) is a counterexample to the argument, and so the argument is invalid.

  6. \(A\textrm{\textbf{a}}B, C\textrm{\textbf{i}}B, C\textrm{\textbf{e}}D\not \models A\textrm{\textbf{o}}D\)

    Let \(a\in \Omega\) and let \(\sigma\) be an interpretation such that: \[ \sigma(A) = \emptyset \]

    \[ \sigma(B) = \{a\} \]

    \[ \sigma(C) = \{a\} \]

    \[ \sigma(D) = \emptyset \]

    On this interpretation:

    \[ \sigma(A) = \emptyset \subset \sigma(B) \hspace{3mm}\Rightarrow\hspace{3mm} v^\sigma(A\textrm{\textbf{a}}B) = \mathbf{T} \]

    \[ \sigma(C)\cap\sigma(B) = \{a\}\cap\{a\} =\{a\} \hspace{3mm}\Rightarrow\hspace{3mm} v^\sigma(C\textrm{\textbf{i}}B) = \mathbf{T} \]

    \[ \sigma(C)\cap\sigma(D) = \{a\}\cap\emptyset = \emptyset \hspace{3mm}\Rightarrow\hspace{3mm} v^\sigma(C\textrm{\textbf{e}}D) = \mathbf{T} \]

    \[ \sigma(A)-\sigma(D) = \emptyset - \emptyset = \emptyset \hspace{3mm}\Rightarrow\hspace{3mm} v^\sigma(A\textrm{\textbf{o}}D) = \mathbf{F} \]

    Hence, \(\sigma\) is a counterexample to the argument, and so the argument is invalid.
